# GitHub Actions 磁盘空间优化方案

## 🚀 快速使用

### 方案选择

| 方案 | 根目录可用空间 | 清理时间 | 说明 |
|-----|---------|---------|---------|
| 🟢 标准型 | ~14 GB | - | 原机器状态，不删除任何软件 |
| 🟡 平衡型 | +40 GB | 2-3分钟 | 删除常见大型软件 (15项) |
| 🔴 极限型 | +54 GB | 3-5分钟 | 删除全部软件 (50+项) |

### 使用方法

**特性：**
- ✅ 自动检出代码仓库
- ✅ 自动执行 start.sh
- ✅ 完全中文化输出
